# Sitio Web para Cliente Real

Este proyecto consiste en el desarrollo de un sitio web para un trabajador independiente, enfocado en la promoción de servicios y captación de clientes.

 Tecnologías utilizadas
- HTML
- CSS
- JavaScript

Objetivo
Desarrollar una solución web funcional, responsive y accesible para mejorar la presencia digital del cliente.

 Demo
https://carlosnino21.github.io/web-servicios-locales/

Características
- Diseño responsive
- Estructura optimizada
- Navegación clara para usuarios
